# LLMO-Content-Generator
KIYONO Internship Assignment - LLMO Content Generator
## プロジェクト概要
KIYONO選考課題として作成する、LLM（大規模言語モデル）に最適化されたコンテンツ生成支援システムです。
従来のSEOではなく、AIに「引用・参照されやすい」コンテンツの自動生成を目指します。

## 開発の背景・着眼点
生成AIが情報検索の主流となる時代において、AIが情報を抽出しやすい構造的かつ信頼性の高いコンテンツ制作を支援します。

## 使用技術（予定）
- Python
- Google Gemini API (gemini-2.5-flash)
- Google Colaboratory
- GitHub (バージョン管理)
